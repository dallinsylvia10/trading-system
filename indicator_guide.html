Signal Logic
How Buy & Sell signals are generated
▲

BUY Signal

Triggered when: Fast EMA crosses above Slow EMA + MACD bullish cross + RSI between 40–65 + Price above 200 EMA (uptrend confirmed). All 4 conditions must align.

▼

SELL Signal

Triggered when: Fast EMA crosses below Slow EMA + MACD bearish cross + RSI between 35–60 + Price below 200 EMA (downtrend confirmed). All 4 conditions must align.

Stop Loss & Take Profit
Stop Loss Formula

ATR-Based SL

SL is placed dynamically using the Average True Range so it adapts to current volatility — not a fixed pip value.

// BUY Stop Loss
SL = close − (ATR × 1.5)

// SELL Stop Loss
SL = close + (ATR × 1.5)
Take Profit Formula

ATR-Based TP

TP is set at 2.5× ATR for a default Risk:Reward of 1:1.67 — adjustable in settings for your trading style.

// BUY Take Profit
TP = close + (ATR × 2.5)

// SELL Take Profit
TP = close − (ATR × 2.5)
Installation Steps

Open TradingView Pine Editor

At the bottom of any chart, click Pine Script Editor. Delete any existing code.

Paste the .pine file contents

Open the downloaded AI_Trading_Indicator.pine file, copy all code, and paste it into the editor.

Click "Add to chart"

Hit the Add to chart button (or Ctrl+Enter). The indicator loads with arrows, SL/TP labels, and an info table.

Set Alerts

Right-click the indicator → Add Alert. Choose "AI BUY Signal" or "AI SELL Signal" and set notification method.

Tune the settings

Click the ⚙️ icon on the indicator to adjust EMA lengths, RSI thresholds, and ATR multipliers to match your asset and timeframe.
